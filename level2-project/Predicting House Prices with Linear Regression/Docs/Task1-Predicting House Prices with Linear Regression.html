7. Model Evaluation

In[39]:
from sklearn.metrics import mean_squared_error, r2_score

# Predict on test set
y_pred = model.predict(X_test)

# Calculate evaluation metrics
mse = mean_squared_error(y_test, y_pred)
r2 = r2_score(y_test, y_pred)

print(f"Mean Squared Error: {mse}")
print(f"R-squared: {r2}")
rmse = mse ** 0.5
print(f"Root Mean Squared Error: {rmse}")

Conclusion:

The linear regression model achieved an R-squared value of 0.65, indicating that approximately 65.29% of the variance in house prices is explained by the model. However, the high Mean Squared Error (MSE) of 1,754,318,687,330.66 and Root Mean Squared Error (RMSE) of 1,324,506.96 suggest that the predictions have a significant average deviation from the actual prices. While the model demonstrates moderate explanatory power, further refinement through feature engineering, data preprocessing, or using more advanced algorithms may be needed to improve predictive accuracy.
